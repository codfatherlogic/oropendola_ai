1. Introduction

Welcome to Oropendola AI. We are committed to protecting your privacy and ensuring the security of your personal information and source code. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our AI-powered coding assistant platform.

Oropendola AI is developed and operated by CODFATHER LOGIC LLP, a limited liability partnership registered in Kerala, India.

Our Core Commitment: Your source code is yours. We do not use your proprietary code to train our AI models or share it with third parties without your explicit consent.

2. Information We Collect

2.1 Account Information

  • Full name and email address
  • Password (encrypted and hashed)
  • Company name and role (optional)

2.2 Usage Data

  • API request metadata and timestamps
  • Feature usage statistics
  • Session duration and frequency
  • Error logs and performance metrics

2.3 Code Metadata

  • Programming languages used
  • File types and extensions
  • Code context (temporary, for AI completions)
  • Repository structure (VS Code extension)

Important: We do not permanently store your source code unless you enable optional codebase indexing.

2.4 Payment Information

  • Billing name and address
  • Payment method (via Razorpay/PayU)
  • Transaction history

Note: Credit card details are processed by PCI-DSS compliant providers only.

2.5 Device Information

  • IP address (country-level geolocation)
  • Browser and operating system
  • Device identifiers (anonymized)
  • VS Code version and configuration

3. How We Use Your Information

3.1 Service Delivery

  • Provide AI-powered code completions and refactoring
  • Route requests to optimal AI models
  • Maintain service performance and reliability

3.2 Account Management

  • Manage accounts and authentication
  • Generate API keys and access tokens
  • Process subscription billing

3.3 Communication

  • Send service updates and security alerts
  • Respond to support inquiries
  • Share product updates (with consent)

3.4 Analytics and Security

  • Analyze usage patterns for improvements
  • Monitor service health
  • Detect and prevent fraud
  • Ensure compliance and security

4. Code Privacy & Security

Zero Training Guarantee: Your proprietary source code is NEVER used to train our AI models or improve services for other users.

4.1 Code Processing

  • Temporary Processing: Code sent to AI providers for real-time completions only
  • TLS 1.3 Encryption: All code encrypted during transmission
  • Immediate Deletion: Code deleted after processing (seconds)
  • Isolated Processing: No cross-user data sharing

4.2 Optional Codebase Indexing

  • Full control over indexed files
  • AES-256 encrypted storage
  • Delete anytime from dashboard
  • Never shared with third parties

4.3 Enterprise Privacy

  • Dedicated AI model instances
  • On-premises deployment options
  • Custom retention policies
  • Advanced audit logging

5. Data Storage and Retention

5.1 Storage Locations

  • Primary: India (Mumbai) - AWS
  • Backup: Singapore - AWS
  • ISO 27001 and SOC 2 certified

5.2 Retention Periods

Data Type Retention Period
